03

How It Works

Microservices Architecture

  • Finder: Autonomous relay discovery via NIP-65 parsing and network crawling
  • Monitor: Health checks, latency measurements, protocol compliance (NIP-11/66)
  • Synchronizer: Parallel event collection with cryptographic validation (secp256k1)
  • Tor Proxy: Complete .onion network coverage for censorship-resistant monitoring
  • PostgreSQL: 600+ GB database with full temporal provenance tracking

Built in Python | Orchestrated with Docker | Licensed under MIT
